## 🛠️ Setup Instructions

### Prerequisites
- Node.js v16+ 
- npm

### Installation
Install Node Package Manager(npm)
Open the file directory in terminal.
```npm install```

### Running the Project
Open the file directory in terminal.
```node app.js```
